# 📱 OmniLight by Abstrackt iOS — Сборка через GitHub Actions

> Этот способ позволяет бесплатно собрать `.ipa` файл на macOS-серверах GitHub, 
> а затем установить его на iPhone через **Windows** с помощью Sideloadly.

---

## Что необходимо

| Инструмент | Цена | Ссылка |
|-----------|------|-----------|
| GitHub-аккаунт | Бесплатно | [github.com](https://github.com) |
| Git для Windows | Бесплатно | [git-scm.com](https://git-scm.com/download/win) |
| Sideloadly (Windows) | Бесплатно | [sideloadly.io](https://sideloadly.io) |
| iTunes для Windows | Бесплатно | [apple.com/itunes](https://www.apple.com/itunes/) |
| Apple ID | Бесплатно | [appleid.apple.com](https://appleid.apple.com) |

---

## Шаг 1 — Создание репозитория на GitHub

1. Войдите на [github.com](https://github.com).
2. Нажмите кнопку **"New repository"** (или перейдите по ссылке [github.com/new](https://github.com/new)).
3. Укажите имя репозитория: `OmniLight`.
4. Видимость: **Public** (это важно, так как для публичных репозиториев минуты GitHub Actions бесплатны и безлимитны).
5. Нажмите **"Create repository"**.

---

## Шаг 2 — Публикация кода на GitHub

Для вашего удобства в папке `github-deploy` подготовлен автоматический скрипт:
1. Запустите файл [push_to_github.bat](file:///d:/OmniLight/github-deploy/push_to_github.bat).
2. Скрипт попросит ввести ваше имя пользователя GitHub и подтвердить создание репозитория.
3. Следуйте простым инструкциям в консоли (потребуется создать Personal Access Token на GitHub вместо обычного пароля).
4. Скрипт инициализирует Git, применит уже настроенный [gitignore](file:///d:/OmniLight/.gitignore), сделает первый коммит и загрузит код в ваш репозиторий.

---

## Шаг 3 — Запуск сборки

Сборка запускается автоматически при отправке кода на GitHub.
Если вам нужно запустить сборку вручную:
1. Перейдите во вкладку **Actions** в вашем репозитории на GitHub.
2. В левой колонке выберите **"Build OmniLight iOS"**.
3. Нажмите кнопку **"Run workflow"** → **"Run workflow"**.

---

## Шаг 4 — Скачивание собранного файла

1. Во вкладке **Actions** выберите последнюю успешную сборку (отмеченную зеленой галочкой ✅).
2. Прокрутите страницу вниз до раздела **"Artifacts"**.
3. Нажмите на имя архива (например, `OmniLight-iOS-[хеш]`) для скачивания `.zip` файла.
4. Распакуйте архив на компьютере — внутри вы найдете файл `OmniLight.app`.

---

## Шаг 5 — Установка `.ipa` на iPhone через Sideloadly

Благодаря обновлению воркфлоу, GitHub Actions теперь автоматически упаковывает проект в файл `OmniLight.ipa`. Вам больше не нужно делать это вручную!
1. Распакуйте скачанный архив `OmniLight-iOS-IPA-N.zip`.
2. Внутри вы найдете уже готовый файл **`OmniLight.ipa`**.

### Установка приложения:
1. Подключите ваш iPhone к компьютеру через USB-кабель.
2. На экране телефона выберите **"Доверять этому компьютеру"** и введите код-пароль.
3. Запустите программу **Sideloadly** на Windows.
4. Перетащите файл `OmniLight.ipa` в окно Sideloadly.
5. В поле **Apple Account** укажите ваш Apple ID (email).
6. Нажмите кнопку **"Start"** и введите пароль от вашего Apple ID.
7. Sideloadly подпишет приложение вашей бесплатной подписью разработчика и установит его на устройство.

### Активация на iPhone:
1. На iPhone перейдите в **Настройки → Основные → VPN и управление устройством**.
2. В разделе "ПО разработчика" выберите ваш Apple ID.
3. Нажмите **"Доверять [ваш Apple ID]"** и подтвердите выбор.

---

## ⚠️ Ограничения бесплатной подписи Apple ID
- Можно установить не более **3 приложений** одновременно.
- Сертификат действует **7 дней**. После этого приложение нужно переустановить (или Sideloadly обновит его автоматически по Wi-Fi, если компьютер и телефон в одной сети).
